Barber Management System

 web application for managing barber shop operations, built with Spring Boot and Java.



   1. Location Hierarchy Management
- **Provinces** → **Districts** → **Sectors** → **Cells** → **Villages**
- Complete Rwandan administrative structure support
- Hierarchical location management with parent-child relationships

  2. User Management
- **Multi-role system**: BARBER, CUSTOMER
- User registration with email, password, and personal details
- Location-based user assignment (village level)
- Complete CRUD operations for users

 3. Service Management
- Service catalog management (Haircut, Beard Trim, etc.)
- Pricing and duration configuration
- Active/inactive service status
- Professional service descriptions

 4. Customer Management
- Customer profile creation
- Preference tracking
- Loyalty points system
- Preferred payment methods

   5. Appointment Management
- Appointment scheduling with barbers
- Service selection and pricing
- Multiple statuses: CONFIRMED, NO_SHOW, COMPLETED
- Automatic end time calculation based on service duration

   6. User Profile Management
- Detailed user profiles with personal information
- Bio and professional descriptions for barbers
- Gender and date of birth management
- Profile picture support

  7. Barber Management
- Barber specialization and expertise
- Rating system
- Years of experience tracking
- License number verification
- Availability status
